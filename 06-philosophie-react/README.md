# Philosophie React

Ce dossier explique les concepts fondamentaux de React.

## Concepts clés

### 1. Composants
React divise l'UI en composants réutilisables et indépendants.

### 2. Déclaratif vs Impératif
- **Impératif** (jQuery) : "Fais ceci, puis cela, puis ça..."
- **Déclaratif** (React) : "Voici à quoi ça doit ressembler"

### 3. Virtual DOM
React maintient une copie virtuelle du DOM et optimise les mises à jour.

### 4. Flux de données unidirectionnel
Les données descendent (props), les événements remontent (callbacks).
