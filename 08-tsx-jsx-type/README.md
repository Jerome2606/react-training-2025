# TSX : JSX avec TypeScript

TSX = JSX + TypeScript. C'est l'extension `.tsx` au lieu de `.jsx`.

## Pourquoi TSX ?

1. **Typage des props** : Erreurs détectées à la compilation
2. **Autocomplétion** : L'IDE connaît les props disponibles
3. **Documentation** : Les types documentent le code

## Fichiers

- `TypingProps.tsx` - Typer les props d'un composant
- `TypingEvents.tsx` - Typer les événements
- `TypingChildren.tsx` - Typer children et ReactNode
- `GenericComponents.tsx` - Composants génériques
