# React Router

Ce dossier contient des exemples d'utilisation de React Router v6.

## Installation

```bash
npm install react-router-dom
```

## Fichiers

- `App.tsx` - Configuration des routes
- `Layout.tsx` - Layout avec navigation
- `pages/` - Composants de pages
- `ProtectedRoute.tsx` - Route protégée

## Concepts clés

- `BrowserRouter` : Provider de routing
- `Routes` / `Route` : Définition des routes
- `Link` / `NavLink` : Navigation
- `useNavigate` : Navigation programmatique
- `useParams` : Paramètres d'URL
- `useSearchParams` : Query params
- `Outlet` : Rendu des routes enfants
