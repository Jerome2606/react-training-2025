# Debug et Performance

Ce dossier couvre les techniques de debugging et d'optimisation des performances React.

## Outils de debugging

### React Developer Tools
- Extension Chrome/Firefox
- Inspecter les composants
- Voir les props et state
- Profiler les renders

### Console et Debugger
- `console.log`, `console.table`, `console.group`
- Breakpoints dans VS Code
- `debugger;` statement

## Outils de performance

### React Profiler
- Mesurer le temps de render
- Identifier les re-renders inutiles

### Lighthouse
- Audit de performance automatique
- Scores et recommandations

- `DebuggingTechniques.tsx` - Techniques de debugging
- `PerformanceOptimization.tsx` - Patterns d'optimisation
- `CommonBugs.tsx` - Bugs courants et solutions
