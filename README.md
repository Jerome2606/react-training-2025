## Prérequis

- Node.js >= 22.x
- npm >= 10.x

## Installation globale

```bash
# Cloner le repo
git clone <repo-url>
cd formation-react-examples

# Pour chaque exemple, naviguer dans le dossier et installer
cd 01-javascript-vs-typescript
npm install
```

## API utilisée pour l'exercice final

L'exercice final utilise l'API **RAWG** (https://rawg.io/apidocs) pour récupérer une liste de jeux vidéo avec leurs images.

Pour obtenir une clé API gratuite :
1. Créer un compte sur https://rawg.io/
2. Aller dans les paramètres du compte
3. Copier votre clé API
