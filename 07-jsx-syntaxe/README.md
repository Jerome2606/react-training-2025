# Syntaxe JSX

JSX est une extension de syntaxe qui permet d'écrire du HTML dans JavaScript.

## Règles fondamentales

1. **Un seul élément racine** (ou Fragment `<>...</>`)
2. **Toutes les balises doivent être fermées** (`<img />`, `<br />`)
3. **className** au lieu de class
4. **camelCase** pour les attributs (`onClick`, `tabIndex`)
5. **Expressions JS** entre accolades `{}`

## Fichiers

- `JsxBasics.tsx` - Syntaxe de base
- `JsxExpressions.tsx` - Expressions et rendu conditionnel
- `JsxLists.tsx` - Rendu de listes avec map
